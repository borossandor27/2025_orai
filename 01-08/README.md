# React routing

Egyszerű navigációs feladat, amelyben három lap között lehet váltani

## Telepítés

```jsx
npm i react-router-dom
```

## BrowserRouter

A `<BrowserRouter>` nyitó és zárótagja között lehet használni a további elemeket. **root** elem!

## Routes

A `<Routes>` elemek között lehet az egyes útvonalakat megadni

## Route

Egy adott megjelenítendő ***lapot*** ezzel tudunk megadni a `to` és `element` tulajdonságok segítségével

## NavLink

Itt adahatjuk meg a felhasználónak felajánlott lapot
